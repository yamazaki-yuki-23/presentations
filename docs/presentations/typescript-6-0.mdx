# TypeScript 6.0

この発表では、TypeScript 6.0 で導入された構造的な変更を扱います。

## 焦点

- 暗黙的な挙動の削減
- ESM ファースト設計への移行
- 実プロジェクトに影響する破壊的変更
- TypeScript 7.0 への準備

## 主なトピック

- `strict` のデフォルト変更
- `types` のデフォルト変更
- `target` / `module` の見直し
- `moduleResolution` の再設計
- `baseUrl` の非推奨化
- `stableTypeOrdering` と移行検証
